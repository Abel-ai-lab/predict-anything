# Contributing to Abel Strategy Research Skills

Thank you for improving Abel Strategy Research Skills.

Good contributions include:

- new strategy discovery workflows
- clearer causal-driver analysis prompts
- better installation docs
- platform-specific fixes for Claude Code, Codex, OpenCode, or ClawHub
- auth and troubleshooting improvements

Please avoid contributions that turn these skills into trade execution, automated trading, or backtesting tools.

## Development Notes

- Source skills live under `skills/`.
- Maintainer-owned render sources live under `maintainers/`.
- Generated release artifacts live under `dist/` and should not be edited manually.
- Normal feature work should not bump source skill versions or top-level release history.
